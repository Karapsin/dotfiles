# Repository Agent Notes

## i3 Shortcut Documentation

When editing `i3/.config/i3/config`, shortcut launch scripts, or app launcher
paths used by i3 bindings, update the `README.md` `Custom i3 Shortcuts`
section in the same change.

Keep that section ordered as:
- app shortcuts first, with Windows analogs for Linux-specific apps when useful
- Polybar shortcuts second
- i3 action shortcuts last

Check the documented shortcut, action text, and grouping against the actual i3
binding before finishing the edit.
